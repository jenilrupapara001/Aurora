## Installation

### Via Shopify Admin
1. Download the theme ZIP file
2. Go to Online Store > Themes in your Shopify admin
3. Click "Upload theme"
4. Select the ZIP file and upload
5. Click "Customize" to configure your theme

### Via Shopify CLI
```bash
shopify theme push --path=/path/to/Aurora
```

## Customization

### Color Palette
Customize your brand colors in Theme Settings > Colors:
- Primary Accent
- Neutral Light/Medium/Dark
- Highlight (for sales and urgency)
- Success (for badges)

### Typography
Choose from Shopify's font library in Theme Settings > Typography:
- Heading Font
- Body Font
- Font Size Base
- Heading Scale

### Layout
Configure global layout settings:
- Max Content Width
- Section Spacing
- Layout Style (Minimal, Editorial, Bold)

## Sections

### Homepage
- **Hero Banner**: Full-width hero with customizable text alignment
- **Featured Collections**: Showcase collection categories
- **Featured Products**: Display trending or featured products

### Product Page
- **Main Product**: Image gallery, variant selector, sticky add-to-cart
- **Product Recommendations**: AI-powered product suggestions

### Collection Page
- **Collection Banner**: Collection title, description, and image
- **Main Collection**: Product grid with sorting and filtering
